# AGENTS.md

Rules for any AI assistant or human working in this repository. These are mandatory and apply at all times.

## Writing style
- Never use em dashes. Use commas, parentheses, colons, or separate sentences instead.
- Never use semicolons in prose, in markdown, or in code comments. Semicolons are allowed only inside uncommented code where the language genuinely requires them (for example CSS).
- Write for fast comprehension. Prefer short, direct sentences.

## Attribution
- Never mention LLM or AI contribution anywhere. Not in commit messages, not in documents, not in code or comments.

## Ignore list
- Always ignore .claude, .specstory, and anything that refers to an LLM or AI tool.

## Project docs
- Keep quick.md and deep.md current as the project progresses.
